# QPower-ISMS ISO 27001 文件生成系統 — 安裝與使用手冊

> 本手冊專為非技術人員設計。只需按照步驟操作，即可在 5 分鐘內開始您的文件生成之旅。

---

## 一、系統快速啟動 (五分鐘上手)

本系統封裝為 `ISO27001_Generator` 資料夾，內置自動化環境設定。

### 1.1 macOS (蘋果電腦)

1. **進入視窗**：開啟電腦上的「終端機」(Terminal)。
2. **定位路徑**：輸入 `cd ` (注意後面有一個空格)，然後將解壓縮後的 `ISO27001_Generator` 資料夾**直接拖入**終端機視窗。
3. **執行啟動**：按下 `Enter` 鍵，接著輸入以下指令並按 `Enter`：
   ```bash
   chmod +x start.sh && ./start.sh
   ```
4. **開始使用**：待顯示啟動成功後，在瀏覽器輸入 `http://localhost:5001`。

### 1.2 Windows (一般 PC)

1. **進入資料夾**：開啟解壓縮後的 `ISO27001_Generator` 資料夾。
2. **啟動系統**：雙擊 **`start.sh`** (需安裝 Git Bash 或 WSL) 或使用終端機執行。
   > [!NOTE]
   > 建議在具備 Bash 環境的系統下運行。

## 二、安全性與系統完整性說明

本系統採用 **資料封裝與完整性校驗技術**，以確保所有文件結構在自動化導出過程中的精確性與一致性。

### 2.1 系統核心素材
為維持生成邏輯的嚴謹性，以下核心素材由系統底層管理：
- **`files/marked/`**：系統導出專用之文件結構素材。
- **`examples/`**：實施參考範例。
- **內部驅動模組**：支撐系統自動化運行之核心程式。

> [!TIP]
> **溫馨提醒**：這些檔案為系統運行必備的「資料來源」，無法直接使用一般文書軟體開啟。這是為了避免手動修改導致系統導出錯誤的校驗機制。您只需透過網頁介面填妥資訊，系統便會自動將這些素材轉化為屬於您的標準 Word/Excel 文件。

### 2.2 檔案結構說明

| 檔案/資料夾 | 說明 |
|---|---|
| `start.sh` | **一鍵啟動腳本**：啟動系統的唯一入口 |
| `安裝與使用手冊.pdf` | 您正在閱讀的這份文件 |
| `output/` | **成果存放目錄**：生成的成果文件將存放在此處 |

## 三、前置環境需求

本系統運行的唯一需求是電腦中需具備 **Python 3.10+** 環境。

- **安裝方式**：若啟動時提示「找不到 python」，請前往 [Python.org](https://www.python.org/downloads/) 下載安裝，安裝時**務必勾選「Add Python to PATH」**。

---

## 四、使用流程引導 (唯一取得文件路徑)

填寫問卷是取得完整 ISMS 文件的**唯一管道**：

1. **基本資料**：設定公司名稱、發布日期、前綴編號。
2. **風險參數**：設定風險門檻、資產分級權重。
3. **程序細節**：設定密碼策略、備份週期等具體管理細節。
4. **勾稽生成**：
   - 系統會自動檢查邏輯一致性。
   - 點擊「⚡ 一鍵生成」按鈕。
   - **完成後，系統會解密範本並填入您的資料，生成您可以自由編輯的 `.docx` 與 `.xlsx` 檔案於 `output/` 資料夾中。**

## 五、常見問題解決 (FAQ)

### Q1：為什麼 `files` 裡面的 Word 檔打不開？
這是正常的。為了確保最終生成文件能精確導出，系統對底層素材進行了結構化封裝。請您務必經由網頁流程完成資訊確認，點擊「生成文件」後，系統會即時為您產出完整的明文文件。

### Q2：如何停止系統？
在終端機視窗中按下鍵盤的 `Ctrl + C`，或直接關閉該視窗。

### Q3：不小心填寫錯了，產出的檔案也要重改嗎？
不用！您只需回到瀏覽器網頁端修改錯誤的欄位，再次按下「一鍵生成」，系統會自動將舊檔案覆蓋更新。
